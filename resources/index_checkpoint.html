Core rules

Term definition

Term Meaning
Unit: health Inherent property of all units: The initial and maximum value for health is 8.
Weapon: hits Number before "x"
Distance between models Shortest distance between any model surface points of both models.
Destroyed Removed from the board space.
Near Within 5cm.
Visibility A model is visible when any of its surface points is visible.
A unit is visible if any of its models are visible.
In cover Drawing the shortest possible line between two units passes through a cover object, and the attacking unit is not near all passed cover objects.
Unit A collection of one or more identical models that act as one entity.
Unit coherency When every model is within 5 cm of another model in the unit.
If at the beginning of any action, unit coherency is not given for any friendly unit, destroy models in that unit until unit coherency is given.


Preparation

Agree on:
  • board space (including cover, special zones, objective markers)
  • a starting player & deployment zone allocation
  • a total army cost limit (example: 1000 points)
  • a game duration limit (number of turns).
Then, every player chooses one faction, and then any number of unit instances of that faction, with a total points cost lower than the agreed limit.

Turns

Every unit starts the game in reserves (outside of the board space).

Players alternate selecting one of their previously unselected units, starting with the starting player.
When a unit is selected, it can sequentially perform up to 2 different actions.
The turn ends when all units have been selected once.

Game end

In order, a winning player is determined by:
  1. victory points
  2. points cost of non-destroyed models

Default actions

Deploy

This action can only be performed when this unit is in reserves.
This unit may be placed anywhere in the owner's deployment zone.

Attack

For every model in the attacking unit:
Select one of the model's weapons, and, if possible, select one visible enemy unit that is either not in cover or within 20 cm as "target unit" for this weapon.
If no target unit can be selected, end the attack sequence for this weapon.

Reduce unit: health by damage.

If unit: health <= 0:
The enemy player must choose to destroy one model in the target unit.
Then, restore the unit health tokens.

Heavy attack

This action must be the first one performed for this unit, and cannot perform the "Attack" action as a second action this turn.
Perform one "Attack" action, and during this attack, can use weapons with
heavy

Restricted actions

Move

Every model in the unit may move up to its value x in centimeters.
Models cannot move over enemy models or more than 5cm vertically.

Fly

Every model in the unit may move up to its value x in centimeters.
Models can move over enemy models and vertically.

Score objective

Cannot perform this action if another action has already been performed by this unit this turn.
If within 5 cm of an objective marker, and no enemy unit is within 5 cm of that objective marker: Gain 1 victory point and remove the objective marker.

Setup teleportation

May set up a teleporter token on the board space, where allowed.
When this unit performs the "Deploy" action: May place the unit as close as possible to the placed teleporter token, and then remove the teleporter token.

Recall

Place this unit into reserves.
